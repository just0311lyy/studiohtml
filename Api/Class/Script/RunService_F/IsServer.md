# IsServer
-----------------------------------------------------------------------------------------
## 描述

函数所属类：[Runservice]()

该函数会返回当前[Runservice]()运行环境，如果实在[Script]()或是被[Script]()调用的[ModelScript]()则返回true，其他情况则是返回false。

-----------------------------------------------------------------------------------------

## 返回

|<div style="width:400px">返回值类型</div>|<div style="width:400px">描述</div>|
|:--------------------|:--------------------|
|   [bool]()  | 返回当前环境是否运行在服务器上 |
