# IsClient
-----------------------------------------------------------------------------------------
## 描述

函数所属类：[Runservice]()

该参数会返回当前[Runservice]()是否运行在客户端中，如果为客户端则会返回true，其他情况则会返回false

-----------------------------------------------------------------------------------------

## 返回参数

|<div style="width:400px">返回值类型</div>|<div style="width:400px">描述</div>|
|:--------------------|:--------------------|
|   [bool]()  | 返回当前环境是否运行在客户端中 |
