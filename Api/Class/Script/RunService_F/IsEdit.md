# IsEdit
-----------------------------------------------------------------------------------------
## 描述

函数所属类：[Runservice]()

该函数会返回当前[Runservice]()是否在编辑模式下，如果在编辑模式返回true，其他情况则会返回false。

-----------------------------------------------------------------------------------------

## 返回

|<div style="width:400px">返回值类型</div>|<div style="width:400px">描述</div>|
|:--------------------|:--------------------|
|   [bool]()  | 返回当前环境是否在编辑模式下 |
