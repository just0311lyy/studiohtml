# Interval
-----------------------------------------------------------------------------------------
## 描述

函数所属类：[Timer]()

定时器间隔，单位为秒，当[Timer.Loop]()为true时，会根据该间隔循环进行。最小精度为1/50秒，最大精度为1000秒。
