# RenderPriority

枚举所属类：[RunService](/Api/Class/Script/RunService.md)

------------------------------------------------------------------------------------------
## 描述:

渲染优先级枚举：
- 优先运行
- 第 2 位运行
- 在 Input（输入）后运行
- 在 Camera（镜头）后运行
- 在 Character 后运行

|名称   |值  |描述|
