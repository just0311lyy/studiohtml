# FillMethod

枚举所属类：[`UIImage`](/Api/Class/Scene/SceneUIImage.md)

------------------------------------------------------------------------------------------
## 描述

填充模式：
- 无，不会应用填充属性；
- 水平填充模式 ；
- 垂直填充模式；
- 以图片上方中点为起点，360度填充模式。
