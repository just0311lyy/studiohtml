# MiniStudio 快捷参考
-----------------------------------------------------------------------------------------

本指南概述了 MiniStudio 的基本功能，这是用于在 MiniStudio 中制作游戏、编写游戏脚本和测试游戏的多合一工具。

## 移动镜头
-----------------------------------------------------------------------------------------

在游戏编辑器窗口中，你可以通过以下控制项来移动镜头：

|<div style="width:150px">**按键/快捷键**</div>|<div style="width:150px">**动作**</div>|<div style="width:500px">**描述**</div>|
|:---|:---|:---|
|`W`|前进|使镜头向前移动|
|`S`|后退|使镜头向后退|
|`A`|左移|使镜头向左移动。|
|`D`|右移|使镜头向右移动。|
|`Q`|下移|使镜头向下移动。|
|`E`|上移|使镜头向上移动。|
|`Shift`|改变镜头速度|结合其他任意移动按键，更改镜头速度（可通过 `Studio` 设置来自定义镜头速度）|
|`F`|聚焦|将镜头聚焦到所选部件上|
|鼠标右键|转动镜头|按住并拖动可以自由地转动镜头视角|
|鼠标中键|平移镜头|按住并拖动可以平移（滑动）镜头|
|鼠标滚轮|缩放|缩放镜头|

如果镜头未响应键盘控制键，请在游戏编辑器窗口中单击以将其激活。
