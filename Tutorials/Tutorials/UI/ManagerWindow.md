# MiniStudio 快捷参考
-----------------------------------------------------------------------------------------

本指南概述了 MiniStudio 的基本功能，这是用于在 MiniStudio 中制作游戏、编写游戏脚本和测试游戏的多合一工具。

## 场景树窗口
-----------------------------------------------------------------------------------------

WorkSpace （管理器）窗口可显示场景中每个对象的层次结构列表。可从 View （视图）选项卡切换。
